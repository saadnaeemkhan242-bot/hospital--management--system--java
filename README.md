 Hospital Management System (Java)

This is a simple Java project demonstrating the concept of **Encapsulation** in Object-Oriented Programming.

 Features
- Private data members (data hiding)
- Getter and Setter methods
- Input validation (age, name, etc.)
- Constructor usage
- Clean and structured code

 Concepts Used
- OOP (Object-Oriented Programming)
- Encapsulation
- Classes and Objects
- Constructors

 Project Structure
- `Patient.java` → Contains patient data and methods
- `Hospital.java` → Main class to run the program

 How to Run
1. Open project in NetBeans or any Java IDE
2. Run `Hospital.java`

 Output Example
Displays patient details like ID, Name, Age, and Disease.
